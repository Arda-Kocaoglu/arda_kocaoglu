# İstatistik ve İhtimaller Teorisi Projesi

## Açıklama
Bu projede verilen veri seti üzerinde istatistiksel hesaplamalar yapılmıştır.

## Yapılan İşlemler
- Ortalama hesaplama
- Varyans ve standart sapma
- Pearson korelasyon katsayısı
- Regresyon analizi
- Hata hesaplamaları (SSE)

## Kullanım
1. Program çalıştırılır
2. Veri seti işlenir
3. Sonuçlar ekrana yazdırılır

## Kullanılan Teknolojiler
- C
- Matematiksel hesaplamalar
